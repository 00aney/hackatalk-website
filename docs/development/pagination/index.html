Pagination

One of the challenges when using graphql is pagination. Facebook's relay cursor pagination seems currently the best solution on implementing infinite scroll view which we normally see in many mobile apps.

Altough it has some limitation when we face offset pagination senario like shown in above image, relay cursor pagination is currently most ideal way to support mobile's infinite scrollview.

Server#

Previously, implementing relay-cursor-pagination with orm like sequelize was bit complicating and here is how we've done in medium post previously.

However, we've migrated to primsa2 and nexus so this becomes much easier. Nexus even supports connection plugin and we'll be using it in our server.
